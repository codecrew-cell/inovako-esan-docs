Training Lab, litoloji siniflandirma modelleri icin veri hazirlama, embedding
cikarma, kumeleme, etiket temizleme, model egitme ve test etme ekranidir.

## Explorer akisi

<Steps>
  <Step title="Kaynak sec ve feature space hazirla">
    Folder, composition veya tile snapshot secip "Prepare Feature Space" calistirin.
    Embedding ilerlemesi canli izlenir.

    <Frame caption="Isleme / progress durumu">
      ![Progress](/assets/screens/training-lab-progress-state-new.png)
    </Frame>
  </Step>
  <Step title="UMAP hesapla ve incele">
    Compute UMAP ile boyut indirgeme ve kumeleme yapilir.

    <Frame caption="Boyutsal projeksiyon haritasi">
      ![Dimensional projection map](/assets/screens/training-lab-dimensional-projection-map.png)
    </Frame>
  </Step>
  <Step title="Cluster'lari yonet">
    Kume isimlerini duzenleyin, lasso ile coklu secim yapin, merge/exclude uygulayin.

    <Frame caption="Kumeleme yoneticisi">
      ![Clustering manager](/assets/screens/training-lab-clustering-manager.png)
    </Frame>
  </Step>
  <Step title="Data Cleanout veya Train'e gec">
    Etiketler netlestikten sonra temizleme veya egitim adimina gecin.
  </Step>
</Steps>

## Kritik endpointler

| Endpoint | Amac |
| --- | --- |
| `POST /traininglab/dataset-info` | Klasor icindeki goruntu sayisini ve durumunu okur. |
| `POST /traininglab/compute-embeddings` | Tile extraction ve embedding surecini baslatir. |
| `GET /traininglab/embedding-progress` | Arka plan islemini izler. |
| `POST /traininglab/compute-umap` | UMAP koordinatlari ve clusterlari hesaplar. |
| `POST /traininglab/recluster` | Kume sayisini degistirir. |
| `POST /traininglab/update-cluster-names` | Cluster/sinif adlarini kaydeder. |
| `POST /traininglab/export-class-package` | Bir sinifin etiket paketini ZIP olarak indirir. |
| `POST /traininglab/import-class-package` | Etiket paketini baska cluster'a uygular. |
| `POST /traininglab/finalize-assignments` | Son etiket atamalarini kalici hale getirir. |
| `GET /traininglab/train-stream` | Model egitim loglarini stream eder. |
| `POST /traininglab/test-sample` | Model test islemini yapar. |

## Sik hatalar

<AccordionGroup>
  <Accordion title="UMAP bos" icon="triangle-alert">
    **Neden:** Embedding tamamlanmadi veya cache yok.

    **Cozum:** Progress bitene kadar bekleyin, sonra Compute Plot.
  </Accordion>
  <Accordion title="Tile snapshot yuklenmiyor" icon="triangle-alert">
    **Neden:** Snapshot READY degil veya composition baglantisi yok.

    **Cozum:** Data Platform'da snapshot durumunu kontrol edin.
  </Accordion>
  <Accordion title="Train modeli kaydetmiyor" icon="triangle-alert">
    **Neden:** Sinifler finalize edilmemis.

    **Cozum:** Data Cleanout/Classes adimlarini tamamlayin.
  </Accordion>
  <Accordion title="Test sonucu beklenenden farkli" icon="triangle-alert">
    **Neden:** Yanlis model veya yanlis trim/window parametresi.

    **Cozum:** Registry metadata ve Settings degerlerini kontrol edin.
  </Accordion>
</AccordionGroup>
