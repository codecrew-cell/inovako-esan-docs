Lithology Editor, kuyu gorsellerinde litoloji bolgelerini model destegiyle
incelemek ve duzenlemek icin kullanilir. Sistem hem entegre API router uzerinden
`/litho/*` endpointlerini hem de gelistirme icin standalone editor servisindeki
`/api/*` endpointlerini destekler.

<Frame caption="Lithology editor">
  ![Lithology editor](/assets/screens/lithology-1.png)
</Frame>

## Adim adim akis

<Steps>
  <Step title="Model sec">
    Kullanilabilir modeller `/litho/models` ile listelenir; litoloji tahmini icin
    kullanilacak modeli secin.
  </Step>
  <Step title="Veri yukle">
    Kuyu klasoru `/litho/load-well` veya mevcut analiz oturumu `/litho/load-session`
    ile yuklenir.
  </Step>
  <Step title="Manevra uret">
    Secilen model ve girdi gorselleri ile `/litho/build-maneuvers` calistirilir.

    <Frame caption="Litoloji bolgeleri ve manevralar">
      ![Lithology regions](/assets/screens/lithology-3.png)
    </Frame>
  </Step>
  <Step title="Incele ve duzenle">
    Bolgeleri, siniflari ve renkleri region editor uzerinde duzenleyin; gorseller
    `/litho/image` ile goruntulenir.

    <Frame caption="Bolge duzenleme">
      ![Lithology editing](/assets/screens/lithology-4.png)
    </Frame>
  </Step>
  <Step title="Kaydet">
    Kullanici duzenlemeleri `/litho/save` ile kaydedilir.
  </Step>
</Steps>

## Ekranda beklenen kontroller

| Kontrol | Kullanimi |
| --- | --- |
| Model listesi | Litoloji tahmini icin kullanilacak modeli secer. |
| Well/session yukleme | Yeni kuyu verisi veya daha once uretilmis session yukler. |
| Region editor | Litoloji araliklarini, siniflarini ve renklerini duzenler. |
| Save | Duzenlenen litoloji manevralarini backend'e kaydeder. |
| Image preview | Birlestirilmis karot gorselini ve duzenlenen bolgeleri kontrol eder. |

## Entegre ve standalone servis farki

| Mod | Endpoint kok dizini | Ne zaman kullanilir |
| --- | --- | --- |
| Entegre uygulama | `/litho` | Normal frontend akisi ve ana backend ile calisma. |
| Standalone editor | `/api` | Lithology editor'u tek basina test etmek veya izole gelistirme yapmak. |
