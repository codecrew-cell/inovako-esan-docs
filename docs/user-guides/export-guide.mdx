Export ekrani, Validate ve Lithology akislari sonrasinda uretilen verileri
raporlanabilir gorsel ve tablo ciktisina donusturur. Kullanici tarafindan
paylasilan nihai dosyalar bu ekrandan alinir.

<Note>
  **On kosullar:** Run tamamlanmis, Validate degisiklikleri kaydedilmis, mineral
  manevralari uretilmis olmali. Litoloji kullanilacaksa Lithology editor ciktisi
  kaydedilmis olmali.
</Note>

## Ana ciktilar

| Cikti | Kaynak | Kontrol |
| --- | --- | --- |
| Manevra gorselleri | `ManueverBlocks` statik klasoru | Kuyu id ve derinlik araliklari dogru mu. |
| Tum dosyalar birlesik gorsel | Backend export klasoru | Goruntu sirasi ve satirlar tutarli mi. |
| CLS birlesik gorsel | Siniflandirma ciktisi | Sinif renkleri beklenen mi. |
| Color change gorseli | Renk analizi | Renk gecisleri mantikli mi. |
| ZIP indirme | Frontend JSZip | Tum beklenen dosyalar pakette var mi. |

## Adim adim akis

<Steps>
  <Step title="Manevra verisini hazirla">
    `get_mineral_maneuvers` veri dondurmezse `mineral_from_changes` ile yeniden uretilir.
  </Step>
  <Step title="On izlemeyi kontrol et">
    Manevra gorsellerini ve tabloyu kuyu id / derinlik araliklari acisindan dogrulayin.
  </Step>
  <Step title="Manevralari kaydet">
    `save_maneuvers` ile ciktiyi kalici hale getirin.
  </Step>
  <Step title="ZIP indir">
    Tum gorseller ve Excel tablolari ZIP olarak indirilir.
  </Step>
</Steps>

## Endpointler

| Endpoint | Amac |
| --- | --- |
| `GET /get_mineral_maneuvers/{session_id}` | Export icin mineral manevra verisini okur. |
| `POST /mineral_from_changes/{session_id}` | Eksik mineral manevrasini yeniden uretir. |
| `POST /getManueverImageNames` | Indirilecek manevra gorsellerini listeler. |
| `GET /download/{hole_id}/{file}` | Tekil gorsel dosyasini indirir. |
| `GET /save_maneuvers/{session_id}` | Manevra ciktisini kalici olarak kaydeder. |
| `/ManueverBlocks/...` | Birlesik export gorsellerini statik olarak sunar. |
| `GET /litho/image` | Litoloji kaynak gorselini preview icin sunar. |

## Kalite kapisi

Export ekrani rapor uretim noktasi oldugu icin, burada gorulen hata genelde
onceki adimdan gelir.
