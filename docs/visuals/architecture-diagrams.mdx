## Lithology sequence

```mermaid
sequenceDiagram
    participant UI as Lithology Page
    participant API as /litho router
    participant DB as Changes/Memory
    participant TL as Training Lab model files
    participant IMG as uploaded_data

    UI->>API: GET /litho/models

    API->>IMG: resolve images
    API->>TL: classify images

```
