# Request / Response Examples

Base URL:

```text
http://localhost:8000
```

## Data upload

`POST /data/upload`

Multipart form alanlari:

| Alan | Tip | Zorunlu | Aciklama |
| --- | --- | --- | --- |
| `dataset_key` | string | Evet | Mantiksal dataset adi |
| `well_name` | string | Evet | Kuyu adi |
| `committed_by` | string | Hayir | Kullanici veya sistem |
| `files` | file[] | Evet | Goruntu, annotation veya artifact dosyalari |

Basarili yanit:

```json
{
  "dataset_key": "YNS-23",
  "snapshot_id": "8f4b9a2c-...",
  "accepted": 120,
  "skipped": 0,
  "duplicate": 3,
  "annotations_accepted": 120,
  "artifacts_accepted": 0
}
```

## Commit dataset snapshot

`POST /data/datasets/{dataset_key}/commit`

```json
{
  "committed_by": "operator",
  "semantic_version": "2026-06-28-baseline"
}
```

Basarili yanit:

```json
{
  "snapshot_id": "8f4b9a2c-...",
  "dataset_key": "YNS-23",
  "semantic_version": "2026-06-28-baseline",
  "parent_snapshot_id": null,
  "image_count": 120,
  "unaffected_models": []
}
```

## Create composition

`POST /data/compositions`

```json
{
  "name": "YNS_23_AND_BN_1084",
  "well_names": ["YNS-23", "BN-1084-15"],
  "created_by": "operator"
}
```

{
  "composition_id": "comp-uuid",

## Lithology load session

`POST /litho/load-session`

```json
{
  "session_id": "session-uuid",
  "hole_id": "YNS-23",
  "model_file": "lithology_xgb_v3.json"
}
```

Basarili yanit ozeti:

```json
{
  "well_id": "YNS-23",
  "folder_path": "uploaded_data/YNS-23",
  "images": [],
  "litho_maneuvers": [],
  "classes": ["andesite", "basalt", "clay"]
}
```

Hata yaniti:

```json
{
  "error": "Litoloji icin geoteknik detections bulunamadi. Once geoteknik analiz/validate gerekli."
}
```

  "images": [
    {
      "filename": "YNS-23_BOX01_0.00-5.70.jpg",
