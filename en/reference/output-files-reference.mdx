# Output Files Reference

## Core analysis outputs

| File or folder | Description |
| --- | --- |
| `uploaded_data/` | Runtime workspace for uploaded source images and folders. |
| `ManueverBlocks/` | Detected maneuver blocks and segment outputs. |
| `ManueverBlocks/div/` | Split or intermediate segment outputs. |
| `TUM_DOSYALAR_BIRLESTIRILMIS.jpg` | Combined main core image. |
| `CLS-TUM_DOSYALAR_BIRLESTIRILMIS.jpg` | Combined image with classification/model output. |
| `TUM_DOSYALAR_BIRLESTIRILMIS_COLOR.jpg` | Combined image with color or metric overlay. |
| `combinedAll.jpg` | Preview output combining multiple images. |

## Training Lab outputs

| Output | Description |
| --- | --- |
| Row crops | Cropped rows from long core imagery. |
| Tiles | Window-based image tiles for embedding and model training. |
| Embedding cache | Feature vectors and UMAP input data. |
| Class package ZIP | Exported class labels and sample package. |
| Model artifact | Saved trained model file. |
| Encoder artifact | Saved feature encoder file. |
| Training history CSV | Training report history. |

## Data Versioning outputs

| Output | Description |
| --- | --- |
| Dataset snapshot | Versioned raw or cleaned dataset state. |
| Artifact record | Metadata for tiles, models, or related files. |
| Composition snapshot | Committed multi-dataset composition state. |
| Tile snapshot | Versioned tile set created from composition or Training Lab. |
| Metadata export | Exported lineage and dataset/model metadata package. |
