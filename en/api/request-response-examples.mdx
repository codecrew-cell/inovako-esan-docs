# Request / Response Examples

Base URL:

```text
http://localhost:8000
```

## Data upload

`POST /data/upload`

Multipart form fields:

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `dataset_key` | string | Yes | Logical dataset name |
| `well_name` | string | Yes | Well name |
| `committed_by` | string | No | User or system |
| `files` | file[] | Yes | Image, annotation or artifact files |

Successful response:

```json
{
  "dataset_key": "YNS-23",
  "snapshot_id": "8f4b9a2c-...",
  "accepted": 120,
  "skipped": 0,
  "duplicate": 3,
  "annotations_accepted": 120,
  "artifacts_accepted": 0
}
```

## Commit dataset snapshot

`POST /data/datasets/{dataset_key}/commit`

```json
{
  "committed_by": "operator",
  "semantic_version": "2026-06-28-baseline"
}
```

Successful response:

```json
{
  "snapshot_id": "8f4b9a2c-...",
  "dataset_key": "YNS-23",
  "semantic_version": "2026-06-28-baseline",
  "parent_snapshot_id": null,
  "image_count": 120,
  "unaffected_models": []
}
```

## Create composition

`POST /data/compositions`

```json
{
  "name": "YNS_23_AND_BN_1084",
  "well_names": ["YNS-23", "BN-1084-15"],
  "created_by": "operator"
}
```

## Lithology load session

`POST /litho/load-session`

```json
{
  "session_id": "session-uuid",
  "hole_id": "YNS-23",
  "model_file": "lithology_xgb_v3.json"
}
```

Successful response summary:

```json
{
  "well_id": "YNS-23",
  "folder_path": "uploaded_data/YNS-23",
  "images": [],
  "litho_maneuvers": [],
  "classes": ["andesite", "basalt", "clay"]
}
```

Error response:

```json
{
  "error": "No geotechnical detections found for lithology. Geotechnical analysis/validate is required first."
}
```

## Lithology build maneuvers

`POST /litho/build-maneuvers`

```json
{
  "session_id": "session-uuid",
  "hole_id": "YNS-23",
  "images": [
    {
      "filename": "YNS-23_BOX01_0.00-5.70.jpg",
      "rows": [
        {
          "row_number": 1,
          "y0": 100,
          "y1": 240,
          "regions": [
            {
              "x0": 20,
              "x1": 360,
              "class_label": "andesite",
              "top_predictions": []
            }
          ]
        }
      ]
    }
  ]
}
```

Response:

```json
{
  "well_id": "YNS-23",
  "session_id": "session-uuid",
  "count": 1,
  "litho_maneuvers": [
    {
      "filename": "YNS-23",
      "mineralBlockStart": 0.0,
      "mineralBlockEnd": 5.7,
      "colorChangeClass": "andesite"
    }
  ]
}
```
