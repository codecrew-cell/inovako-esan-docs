# Validate Screen Guide

## Purpose

Validate is used to inspect model detections and correct them before mineral, lithology, or export steps.

## Main actions

| Action | Description |
| --- | --- |
| Review boxes | Confirm that each detection is correctly positioned. |
| Add box | Create a missing detection. |
| Delete box | Remove an incorrect detection. |
| Change class | Assign the correct class to a detection. |
| Save changes | Persist session changes to the backend. |

## Recommended workflow

1. Move through images in order.
2. Correct obvious false positives and missing detections.
3. Save changes regularly.
4. Continue to Mineral Validate after geotechnical detections are stable.

## Quality checklist

- No important region is missing.
- Incorrect boxes are removed.
- Class labels follow the project standard.
- Bulk save completed successfully.
