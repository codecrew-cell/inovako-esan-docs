# User Walkthrough

## End-to-end path

1. Log in with a valid username.
2. Review Dashboard status and recent activity.
3. Open Run and load a core image folder.
4. Select the required models.
5. Start the analysis and wait for progress completion.
6. Review detections in Validate.
7. Review mineral and lithology information.
8. Export the final package.

## What users should verify

| Stage | Check |
| --- | --- |
| Login | Correct username and role are active. |
| Run | The selected folder is the intended well or core dataset. |
| Models | Model names match the project standard. |
| Validate | Detection boxes are positioned correctly. |
| Mineral Validate | Mineral classifications and changes are saved. |
| Lithology | Lithology regions align with the image. |
| Export | Preview and downloaded output match expected content. |

## When to stop and ask for support

- Model list is empty.
- Uploaded images do not appear.
- Progress stops without a clear error.
- Validate cannot load the processed frame.
- Export file is missing expected images or tables.
