# Turkce Dokumantasyon

Bu alan mevcut Turkce dokumantasyon agacina hizli giris saglar. Ust menuden **English** secilerek Ingilizce dokumantasyon girisine gecilebilir.

## Hizli erisim

Sol menudeki Turkce gruplar mevcut tam dokumantasyon agacini icerir:

| Grup | Icerik |
| --- | --- |
| Baslangic | Lokal kurulum, proje haritasi ve genel giris. |
| Klavuzlar | Operator, gelistirici ve kullanici akislari. |
| Ekran Kilavuzlari | Dashboard, Run, Validate, Export, Training Lab ve admin ekranlari. |
| Mimari ve Is Akislari | Backend, frontend, sistem ve domain akis aciklamalari. |
| API | Backend endpoint gruplari ve ornek payloadlar. |
| Veri ve DB | Storage, database schema ve data versioning operasyonlari. |
| Operasyon | Konfigurasyon, troubleshooting, release ve session recovery. |

## Not

Mevcut Turkce sayfalar eski URL yapisini korur. Bu sayede daha once paylasilmis linkler kirilmaz.
